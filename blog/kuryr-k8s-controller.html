Kuryr Kubernetes Controller

What is Controller in Kubernetes?

Controller implements control loop, watching the shared state of the cluster via Kubernetes API server and makes attempts to move the current state towards desired state.

What is Kuryr Kubernetes Controller in Kubernetes?

Kuryr connects OpenShift with OpenStack in a way that it is watching Kubernetes API server for specific resources events (such as Pods, Services) and then it triggers action to create OpenStack resources (Network, orts)

At the moment, of writing this blog post.
Kuryr is Event-driven.
